Jetbrains released a Google App Engine plugin for IntelliJ

Jetbrains released a Google App Engine plugin for their IntelliJ IDE. I did not see any announcement about this at all. You will have a new option when creating a new project. After supplying the path to the local app engine SDK your almost good to go. I needed to add the app engine jars as libraries to my project. That being out of the way you can start coding, run you local server (a pre configured run configuration is supplied) and also deploy it to Google all from within IntelliJ. ... Read more
