IntelliJ Idea Plugin Property Sorter updated

I’m pretty busy at work right now but I could find some time to add one more feature to my little property sorter plugin for IntelliJ Idea. Comments above property entries will now be preserved when you sort the file. The plugin is available from within Idea or at Google Code.
