Wardley Mapping Canvas

Sep 4, 2019 - 1 min

I’m a fan of Simon Wardley’s Maps.

Ben Mosior over at Miro came up with a nice canvas for Wardley mapping. I could not find a vector version for printing out, so I made one with Affinity Designer.

They are available at my Github Repository. It contains the original .afdesign file as well as a PDF version with and without explanation.
